# 📘 Array Practice Questions (45 Problems)

This repository contains a collection of **45 important array problems** designed to strengthen problem-solving skills in Data Structures and Algorithms (DSA).

These questions range from **basic operations** to **advanced patterns** frequently asked in coding interviews.

---

## 🚀 Topics Covered

- Basic array operations (sum, average, min, max)
- Searching and counting
- Sorting techniques
- Array transformations (shift, rotate)
- Duplicate handling
- Frequency counting
- Subarray problems
- Two-pointer and sliding window techniques
- Advanced problems (Kadane’s Algorithm, Majority Element)

---

## 📋 Problem List

### 🔹 Basic Operations
1. Find the sum of all elements in an array  
2. Find the average of array elements  
3. Find the maximum/minimum element  
4. Search for an element (Linear Search)  
5. Count occurrences of an element  
6. Replace all occurrences of an element  
7. Check if array contains a given number  

### 🔹 Counting & Separation
8. Count even and odd elements  
9. Separate even and odd elements  
10. Copy array elements  
11. Merge two arrays  
12. Compare two arrays  

### 🔹 Index & Shifting
13. First occurrence of an element  
14. Last occurrence of an element  
15. Left shift array  
16. Right shift array  

### 🔹 Sorting & Order
17. Sort in ascending order  
18. Sort in descending order  
19. Second largest element  
20. Second smallest element  

### 🔹 Duplicates & Frequency
21. Remove duplicates  
22. Count frequency of elements  

### 🔹 Array Manipulation
23. Reverse array (in-place)  
24. Rotate array right by k  
25. Rotate array left by k  

### 🔹 Advanced Selection
26. Largest three elements  
27. Smallest three elements  
28. Sum of even and odd separately  
29. Replace negatives with zero  
30. Replace zeros with ones  

### 🔹 Mathematical Problems
31. Difference between max and min  
32. Find pairs with given sum  
33. Check if array is sorted (ascending) 
34. Count elements greater than a given number  

### 🔹 Rearrangement Problems
35. Move zeros to end  
36. Alternate positive and negative numbers  

### 🔹 Important Interview Problems
37. Majority element (> n/2 times)  
38. Intersection of two arrays  
39. Union of two arrays  
40. Missing number (1 to n)  
41. Find duplicate number  
42. Element appearing once (others twice)  

### 🔹 Subarray Problems
43. Maximum subarray sum (Kadane’s Algorithm)  
44. Count subarrays with given sum  
45. Longest subarray with distinct elements
